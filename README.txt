Kuraddo Generator
=================

Kuraddo (クラッド) Generator allows you - web developer - to creates, manages, builds 
and test Spring Boot Applications - this applications can be REST APIs or Web 
Aplication using some template engine (Thymeleaf, Groovy, Mustache, FreeMarker, etc).

In another words, develop CREATE - READ - UPDATE - DELETE functionalities in 3 minutes 
like Ramen Noodles. Please read the documentation in Documentation/ subdirectory in 
order to understand how to install, how to or access the docummentation in the 
official website:
  (In constructor) https://www.leitess.github.io/kuraddo/docs
