Kuraddo Generator
=================

Kuraddo (クラッド) Generator allows you as a web developer to create, manage, build 
and test Spring Boot Applications - These applications can be REST APIs or Web 
Aplication using some template engine (Thymeleaf, Groovy, Mustache, FreeMarker, etc).

In other words, it develops CREATE - READ - UPDATE - DELETE functionalities in 3 minutes 
like ramen noodles. Please read the documentation in Documentation/ subdirectory in 
order to contribute for Kuraddo's development.
